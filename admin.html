GitHub connection
Where to get a token:
  1. Go to github.com/settings/personal-access-tokens/new
  2. Set Repository access to "Only select repositories" → choose this repo.
  3. Under Permissions → Repository permissions, set "Contents" to Read and write.
  4. Generate the token and paste it above. Treat it like a password — anyone with it can edit the site.
This page is public (the repo is public), so the password screen only keeps casual visitors out — the token is what actually protects saving. Don't share your token, and revoke it on GitHub if you ever suspect it leaked.
